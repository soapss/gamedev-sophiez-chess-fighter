# AGENTS Guide

This repository is a Game Dev track submission. Keep the game small, playable,
and easy for agents and reviewers to run.

## Expectations

- Keep `task.toml`, `PRD.md`, and `SETUP.md` in sync.
- Keep screenshots and binary paths referenced by `task.toml` valid.
- Prefer a complete core loop over unfinished extra mechanics.
- Document build and run commands clearly enough for a reviewer to start the
  game in under five minutes.
- Do not commit secrets, generated dependency folders, or large build outputs
  unless the track explicitly requires the artifact.
